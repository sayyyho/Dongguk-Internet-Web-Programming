웹의 태동

인터넷이 등장하면서 세계는 급격하게 변했다고 해도 과언이 아니다. 하지만 어려웠기 때문에 대중에게 풀려도 제대로 사용되지 않았던 인터넷을 그나마 사용하기 편했던 웹이 더 확실하고 빠르게 보급되었다.

월드와이드웹(WWW)

웹이 처음 등장한 이래로 많은 변화가 있었지만 처음에는 초라한 형태였는데, 팀 버너스리는 웹을 고안하면서 웹에서 보여질 문서의 규격도 같이 고안했다. 그게 바로 HTML이라는 언어였고, html 문서는 HTML로 작성이 되고 이렇게 작성된 파일은 웹 브라우저를 통해서 열람이 가능했다.

자바스크립트의 등장

HTML과 CSS 위주의 정적이었던 웹 페이지에서 동적인 요소들이 부각되기 시작하면서 자바스크립트의 영역이 점점 더 커지게 되었고, 웹 사이트에 동적인 요소를 추가하고 자바스크립트를 적극 활용하기 시작했다. 오늘날 웹의 대표 세가지 요소는 HTML, CSS, JavaScript가 되었다.
